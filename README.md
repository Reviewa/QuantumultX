## 📁 仓库结构说明

| 文件夹 | 内容描述 |
|--------|----------|
| [`script/`](./script) | 解锁订阅、会员、广告净化等 JS 脚本（支持 QX/Surge/Loon） |
| [`surge/`](./surge)   | Surge 专用 `.sgmodule` 模块，包含规则、图标配置 |
| [`icon/`](./icon)     | 精美图标，支持 Surge / QX，统一样式更美观 |
| [`loon/`](./loon)     | Loon 平台专属脚本与配置文件 |
| [`ql/`](./ql)         | 青龙面板专用脚本，适配定时任务 / 签到 / 多账号等场景 |
