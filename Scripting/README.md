## ✨ 特性

- 🧩 **模块化设计** – 按功能拆分脚本，通过 `importModule` 或自定义加载器动态引入。
- 🌐 **远程导入** – 直接通过 URL 加载云端模块，无需手动下载到本地。
- 🧠 **自定义扩展** – 提供标准化的模块接口，你可以轻松编写、分享自己的模块。
- 🔁 **本地缓存** – 远程模块自动缓存，支持离线使用和版本控制。
- 📱 **专为 Scripting 优化** – 兼容 Scripting 的文件管理、面板、图表等 API。
